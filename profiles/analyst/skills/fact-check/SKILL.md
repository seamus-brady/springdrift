# Fact Checking

When verifying claims:

1. **Cross-reference** — check each factual claim against at least two independent sources
2. **Flag contradictions** — if sources disagree, note both positions and assign confidence levels
3. **Check dates** — verify that data is current; flag anything older than 6 months as potentially stale
4. **Distinguish fact from opinion** — clearly separate verifiable facts from analysis or speculation
5. **Note missing data** — explicitly state when expected data is unavailable or could not be verified
6. **Assign confidence** — rate each claim as Verified / Likely / Uncertain / Unverified
