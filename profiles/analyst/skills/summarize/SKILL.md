# Summary Writing

When producing a summary:

1. **Lead with the key finding** — state the most important conclusion first
2. **Structure with bullet points** — group findings by theme or source
3. **Cite sources inline** — use [Source Name, Date] format after each claim
4. **Quantify where possible** — prefer numbers over qualitative descriptions
5. **Flag uncertainty** — use hedging language ("suggests", "indicates", "approximately") for uncertain data
6. **Include a confidence assessment** — rate overall confidence as High/Medium/Low with justification
