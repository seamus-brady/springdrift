# Citation Standards

All reports must follow these citation rules:

1. **Inline citations** — place citations immediately after the claim: "Property prices rose 5.2% [CSO, Jan 2026, https://cso.ie/...]"
2. **Source list** — include a "Sources" section at the end listing all referenced materials
3. **Date accessed** — include when the source was accessed if it is a web resource
4. **Primary sources preferred** — cite original data sources over secondary reporting where possible
5. **No fabricated URLs** — if the exact URL is unknown, cite the source name and date only
6. **Attribution for analysis** — when presenting analysis or forecasts, attribute to the originator
